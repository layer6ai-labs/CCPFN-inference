# Data Access

The `example_usage.ipynb` notebook uses the Warfarin dataset from the International Warfarin Pharmacogenetics Consortium (IWPC). This dataset is not included in the repository and must be downloaded separately.

## Steps

1. Download the data from [ClinPGx Downloads](https://www.clinpgx.org/downloads). Locate the **IWPC dataset (.xlsx)** under the "International Warfarin Pharmacogenetics Consortium (IWPC) Data" section.
2. Open the file and export the **"Subject Data"** sheet as a CSV file.
3. Update the file paths in `example_usage.ipynb` to point to your new CSV file.
